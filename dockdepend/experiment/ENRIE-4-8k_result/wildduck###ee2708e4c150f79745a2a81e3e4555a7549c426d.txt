 (7, 8),  # RUN npm install依赖于COPY，因为它需要COPY复制过来的package.json文件
 (8, 9),  # 第二个COPY依赖于前面的RUN npm install，因为它可能复制的是构建后的文件或配置
 (9, 10), # ENV依赖于前面的COPY，因为它可能设置的环境变量与复制过来的文件有关
 (10, 11), # ENTRYPOINT依赖于ENV，因为它可能需要用到ENV设置的环境变量
 (11, 12)] # CMD依赖于ENTRYPOINT，因为它定义了容器启动后要执行的命令，该命令可能依赖于ENTRYPOINT设置的启动方式
