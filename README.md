# DockDepend

<p style="text-indent: 2em;">DockDepend是一个依赖关系抽取工具，能够准确、高效地判断Dockerfile指令行间的依赖关系，为Docker镜像的构建和优化提供有力支持。
DockDepend为不同指令类型提供了统一的表示方式，有效地将指令行的语义信息编码为可计算的数据结构，从而提升了依赖关系判断的准确率和效率。</p>
